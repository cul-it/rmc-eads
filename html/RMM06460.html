COLLECTION DESCRIPTION

Printing equipment and supplies, including 18th century pattern, French style, type mould, 18 point body; ladle; slotted screwdriver; punch, letter P; strike, from P punch; fitted matrix, from P punch; cast sorts, using new mould. Also, matrices for upper- and lower-case letters, a carved wood block, copper plates for engravings, a composer's stick, galley, chase, type cases, a passage of text set up in type, screens for use in hand paper making, and a book binding loom. Also, two metal sheets used in offset printing for the August 16, 1968 issue of a newspaper, the Ithaca Journal.
