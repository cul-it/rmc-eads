COLLECTION DESCRIPTION

The Gay Media Task Force was created by the National Gay Task Force in 1972 to serve as a resource organization for network television programming as it addressed gay issues. The collection includes correspondence, press releases and other information about Newton Deiter, coordinator of the Task Force; film and television scripts on which the Task Force worked; erotica; and 3/4" video tapes of network television programs and source material.
